## Installation :electric_plug:
In order to clone and install So Fresh, So Green, to run locally: 

```md
npm install
```
Then 
* `npm run seed`
Run the following command to start MySQL server
* `npm run dev`


## Usage :gear:
```
This app utilizes:
-Stripe
-Redux
-AWS for deployment and image upload
-3JS(spline)/Tailwind for UX/UI
-JWT Authentication
-Javascript
-Apollo/GraphQL
-DB Mongo/Mongoose
```
